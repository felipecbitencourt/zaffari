# Teste de Markdown

Esta é uma página de teste para validar a implementação do **Nível 2 de Adaptabilidade**.

## O que mudou?
Agora é possível criar páginas usando **Markdown** em vez de HTML puro. Isso facilita muito a edição para quem não é programador.

### Lista de Vantagens
- Mais simples de ler e escrever
- Menor chance de quebrar o layout
- Foco no conteúdo

### Links e Formatação
Podemos usar [links](http://www.google.com) e *itálico* também.

> "A simplicidade é o último grau de sofisticação." - Leonardo da Vinci

Se você está vendo isso formatado (Títulos grandes, negrito, lista), o parser está funcionando!
